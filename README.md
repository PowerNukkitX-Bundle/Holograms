# Holograms
Add holograms to your PowerNukkitX Server

This plugin allows you to add floating text to your PowerNukkitX server.
This plugin is fully UI controllable.

Open the holograms UI with /holograms

There you can create and modify holograms.


This plugin supports [PlaceholderAPI](https://github.com/PowerNukkitX-Bundle/PlaceholderAPI). 
For example, you can write %player% in a line so the players username is displayed in the hologram.
